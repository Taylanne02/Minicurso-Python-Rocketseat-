Minicurso-Python-Rocketseat
Este repositório reúne os aprendizados e práticas desenvolvidos no curso da Rocketseat, com foco na construção de aplicações web utilizando Python e Flask, desde os conceitos iniciais até o deploy da aplicação. Ao longo do curso, trabalhei os principais fundamentos necessários para o desenvolvimento backend, entendendo na prática como estruturar e publicar uma aplicação.

Conteúdos abordados:

Fundamentos do Python
Conceitos de API e arquitetura de aplicações
Métodos HTTP (GET, POST, PUT, DELETE)
Integração com Banco de Dados
Desenvolvimento de aplicações com Flask
Autenticação de usuários com Flask-Login
Tratamento de erros e boas práticas
Deploy da aplicação
Noções básicas de AWS, incluindo conceitos iniciais sobre ambientes, aplicações e autorizações de usuários
Objetivo do curso: Consolidar uma base sólida em desenvolvimento backend com Python, compreendendo o fluxo completo de uma aplicação: criação, autenticação, organização de ambientes e publicação.
